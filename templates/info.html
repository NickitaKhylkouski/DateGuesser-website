Game Information & Features

The Weekday Guessing Game is a fun and educational tool to help you practice mental date calculation skills. The game offers multiple modes to challenge yourself and improve your ability to determine the day of the week for any given date.

Keyboard Shortcuts
  • Space - Start game
  • R - Restart game
  • 1-7 - Select weekday (Monday through Sunday)
  • 1-9,0,-,= - Select month number (1-12) in Month Number game

Practice determining weekdays for dates in the current year. This mode is perfect for beginners as it:

  • Only shows month and day (year is fixed to current year)
  • Allows unlimited time per guess
  • Tracks accuracy and average response time

This mode helps you build fundamental pattern recognition for dates within a single year.

Challenge yourself with dates from any year between 1700 and 2100. Features include:

  • Customizable year range selection
  • Full date display (month/day/year)
  • Unlimited time per guess
  • Great for practicing century calculations

Practice the Doomsday Algorithm by determining the anchor day for any year. This mode:

  • Shows only the year
  • Helps master the century anchor days
  • Builds speed in year calculations
Century Anchor Days:
  • 1700s: Sunday
  • 1800s: Friday
  • 1900s: Wednesday
  • 2000s: Tuesday

A fast-paced challenge to test your speed and accuracy. Features:

  • 60-second time limit
  • Rapid-fire date calculations
  • Score tracking for correct answers
  • Available in both current year and custom year range modes

Perfect for testing and improving your calculation speed!

Practice associating month names with their numbers. Features:

  • 60-second time limit
  • Shows month names, guess their numbers
  • Keyboard shortcuts (1-9,0,-,= for months 1-12)
  • Great for building quick month recognition

General Tips:
  • Start with the Current Year game to build confidence
  • Use keyboard shortcuts to improve your speed
  • Practice the Doomsday Algorithm basics before attempting custom year ranges
  • Pay attention to your average response times and try to improve them
Learning Path:
  1. Master the Month Number game first
  2. Practice Current Year mode until comfortable
  3. Learn the Doomsday Algorithm basics
  4. Challenge yourself with Custom Year Range
  5. Test your speed with Date Rush
